Login POST

  • URL:

    /api/auths/login

  • email:

    Debe ser un correo valido

  • password:

    Debe tener mas de 8 digitos

Conexion activa POST

  • URL:

    /api/auths/conextion/active/:id

Conexion inactiva POST

  • URL:

    /api/auths/conextion/desactive/:id

Crear cuenta POST

  • URL:

    /api/auths/add

  • avatar:

    Debe ser una imagen

  • email:

    Debe ser un correo valido

  • password:

    Debe tener mas de 8 digitos

  • name:

    Debe ser nombre

  • lastname:

    Debe ser apellido

  • typeUser:

    Debe ser alumno o docente

  • state:

    Debe ser 0 o 1

  • carrier:

    Debe ser un string con el id de la carrera

Eliminar cuenta DELETE

  • URL:

    /api/auths/delete

  • id:

    Debe ser un id

Editar usuario POST

  • URL:

    /api/auths/edit/:id

  • NOTA:

    Lo mismo para crear cuenta con un id
